# @wasser-ui/vue

Vue renderer package for `VerifiedSchemaUI`.

This package provides:

- `WasserRenderer`
- `WasserStreamRenderer`
- `WasserRuntimeProvider`
- `NodeRenderer`
- `SlotRenderer`
- recursive node and slot rendering
- runtime data and action binding
- render diagnostics
- safe fallback rendering

`@wasser-ui/vue` renders verified UI only. Pass raw model output through
`@wasser-ui/core` verification before rendering.
