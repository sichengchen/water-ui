# @wasser-ui/angular

Angular renderer package for `VerifiedSchemaUI`.

This package provides:

- `WasserRendererComponent`
- `WasserStreamRendererComponent`
- `NodeRendererComponent`
- `SlotRendererComponent`
- `WasserOutletComponent`
- `provideWasserRuntime`
- recursive node and slot rendering
- runtime data and action binding
- render diagnostics
- safe fallback rendering

`@wasser-ui/angular` renders verified UI only. Pass raw model output through
`@wasser-ui/core` verification before rendering.
