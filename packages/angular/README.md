# @water-ui/angular

Angular renderer package for `VerifiedSchemaUI`.

This package provides:

- `WaterRendererComponent`
- `WaterStreamRendererComponent`
- `NodeRendererComponent`
- `SlotRendererComponent`
- `WaterOutletComponent`
- `provideWaterRuntime`
- recursive node and slot rendering
- runtime data and action binding
- render diagnostics
- safe fallback rendering

`@water-ui/angular` renders verified UI only. Pass raw model output through
`@water-ui/core` verification before rendering.
