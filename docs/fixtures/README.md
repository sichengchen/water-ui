# Fixture Map

Fixtures are grouped by protocol and package area.

Core registry fixtures:

```txt
docs/fixtures/core/registry/
```

Protocol fixtures:

```txt
docs/fixtures/protocol/documents/
docs/fixtures/protocol/patches/
docs/fixtures/protocol/streams/
```

Verification fixtures:

```txt
docs/fixtures/verification/documents/
```

Prompt golden fixtures live at repo root:

```txt
goldens/prompts/
```

Fixture naming convention:

- `*.valid.json`
- `*.invalid.json`
- `*.valid.jsonl`
- `*.invalid.jsonl`
- `*.registry.ts`
- `*.system.txt`
